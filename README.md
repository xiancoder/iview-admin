<h1> iView Admin - 延伸扩展版本</h1>
<h3>Vue.js 2.0 admin management system template based on iView.</h3>


## 获取最新版本
```bush
# 拷贝项目
git clone https://github.com/xiancoder/iview-admin.git

// 安装依赖(推荐使用淘宝cnpm)
cnpm install

// 开发模式
npm run dev

// 生产模式(带console调试)
npm run build

// 生产模式(不带调试)
npm run online
```
